# Sunlit Patrol

A small Three.js shooter prototype using CC0 low-poly assets. It is the first playable slice for a Summer Afternoon-inspired web 3D shooter: third-person camera, animated player, blaster, targets, HUD, and a lightweight stylized world.

## Run

```bash
npm install
npm run dev
```

The dev server opens on `http://localhost:5173/` by default.

## Build

```bash
npm run build
```

## Assets

- Kenney Blaster Kit: `assets/raw/kenney/blaster-kit`
- Quaternius Ultimate Animated Character Pack sample: `assets/raw/quaternius/ultimate-animated-character`

Both source packs are marked CC0/public domain by their creators. Local license notes are kept in `assets/raw/_licenses`.

Runtime assets used by the app are copied into `public/assets`.
